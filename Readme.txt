Circle-Inspired Optimization Algorithm (CIOA) is a metaheuristic optimization algorithm.

Developed by: Otávio Augusto Peter de Souza (otavio.souza@ufrgs.br)
              Letícia Fleck Fadel Miguel (letffm@ufrgs.br)

Federal University of Rio Grande do Sul (UFRGS), Postgraduate Program in Civil Engineering (PPGEC)

To use the CIOA:

1. Define your objective funcion, constraints and penalty in Objective_Function_CIOA.m file
2. Adjusts the parameters of the algorithm and of the optimization problem in Main_Program_CIOA.m file
3. Run the code in Main_Program_CIOA.m file
